# Power-BI-Mobile-Sales-Dashboard

Mobile Sales Data Analysis Dashboard | Power BI

I’m excited to share my Mobile Sales Data Analysis Dashboard built using Microsoft Power BI as part of my Data Analytics learning journey.

This dashboard analyzes mobile sales performance across different cities, brands, payment methods, and time periods to generate meaningful business insights.

🎯 Project Objective

The objective of this project is to analyze mobile sales data and understand sales trends, customer behavior, and product performance across different cities, brands, and time periods.

🛠 Tools & Technologies Used

Power BI

Excel / CSV Dataset

Power Query (Data Cleaning & Transformation)

Data Visualization Techniques

DAX (Data Analysis Expressions)

Interactive Dashboard Design

📌 Dashboard Features

📊 Built an interactive dashboard using Microsoft Power BI

📈 Displays key KPIs: Total Sales, Total Quantity, Transactions, Average Price

🌍 City-wise sales analysis using map visualization

📱 Brand and mobile model performance comparison

💳 Payment method analysis (UPI, Debit Card, Credit Card, Cash)

⭐ Customer rating analysis (Good, Average, Poor)

📅 Time-based analysis (Monthly, Quarterly, MTD, Same Period Last Year)

🎛 Interactive filters for Brand, Model, Year, Month, and Payment Method

📈 Key Insights

Identified the top-selling mobile brands and models.

Compared current sales with the same period last year.

Analyzed monthly sales growth and trends.

Observed most used payment methods by customers.

Identified cities generating the highest sales revenue.

This project helped me strengthen my skills in data visualization, dashboard design, and business analytics using Power BI.
